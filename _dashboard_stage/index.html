Installer l’app

Installe le dashboard sur ton téléphone ou ton ordinateur pour l’ouvrir en plein écran et recevoir les notifications push de nouvelles inscriptions et commandes.

Le navigateur prépare l’installation. Laisse le dashboard chargé quelques instants si le bouton n’est pas encore prêt.

Sur iPhone, l’installation passe par le menu Partager de Safari. Sur Android et desktop, le navigateur peut proposer une installation native en un clic.
